# Ocean Health Index for the US Northeast

This repository contains the data prep for the US Northeast Ocean Health Index assessment.

Learn more at [ohi-northeast.org](http://www.ohi-northeast.org/)

`prep` folder contains all code and data for each goal layer
`spatial` folder contains spatial data that is used across multiple goals as well as the data used to create the 11 sub-regions
`src` contains common scripts with common functions and objects used throughout the data preparation process
