## Iconic Species sub-goal

This folder describes the methods used to prepare data for the Iconic Species sub-goal.

This folder describes the methods used to prepare the data layers for the Fishing Engagement sub-goal.

Click on the layers to see data preparation:

### [Iconic Species Status](https://ohi-northeast.github.io/ne-prep/prep/sop/ico/iconic_species.html)

- `figs` folder contain figures
- `data` folder contains data generated by scripts in this folder
